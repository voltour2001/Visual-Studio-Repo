Detailed Description

template<class T>
class rt::Soupiterator< T >

rtSoup custom iterator class.

It is a forward iterator based on the STL input iterator std::iterator<std::input_iterator_tag, T> and derived from the corresponding iterator of the rtAccelerationStructure. The iterator has also the alias rtSoup<T>::iterator in complience with the STL iterators for containers.
