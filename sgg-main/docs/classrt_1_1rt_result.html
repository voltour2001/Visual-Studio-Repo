Detailed Description

rtResult is a helper class to accommodate the result code of an operation along with an (optional) result message.

Most non-trivial operations in the Registration Toolkit return a new instance of this class to signify either the successful termination of the operation or the error code and corresponding explanation of the error. The error code is contained in the code public member variable and the corresponding explanation for the returned result in the message member variable.

Examples:
Example-DistanceField.cpp, Example-PLYReader.cpp, and Example-PLYWriter.cpp.

Member Enumeration Documentation

enum rt::rtResult::rt_result_code_t

The result code enumeration type.

The return code type provides a brief description of the nature of the result of an operation. More details are given in the message member variable of the rtResult class.

Enumerator
RT_OK 

Operation terminated normally. No errors.

RT_INVALID_PARAMS 

One or more of the parameter values was invalid.

RT_INVALID_OPERATION 

The call to a function could not be completed since dpendent data were not properly initialized.

RT_INTERNAL_ERROR 

Operation was terminated due to an unexpected (internal) error.

RT_CANCELLED 

Operation was cancelled by the calling function.

RT_FAILURE 

Operation failed to produce a valid result.

RT_END_OF_DATA 

Operation terminated due to lack of sufficient data.
