Detailed Description

template<typename T>
class rt::ads::rtIteratorDF< T >

Dummy distance field iterator. Not used, but must exist.

The distance field is not an iterable data structure, as it makes no sense to get back its member data. Even the inserted elements are not retained but rather only update information on the discretized cells upon insertion.
