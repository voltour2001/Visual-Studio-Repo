float rt::metricEuclidean ( const T &  p1,
const T &  p2

Calculates the Euclidean (L2) norm between two arbitrary vectors of type T and size N.

The input type T must have the [] operator implemented to access its N coordinates. As this is a generic version of the Euclidean distance, it is not optimized for 3D vectors specifically. However, there is a template specialization implemented for Eigen::Vector3f, which is.
