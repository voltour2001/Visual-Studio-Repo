Detailed Description

template<class T>
class rt::ASiterator< T >

rtAccelerationStructure custom iterator class.

It is a forward iterator based on the STL input iterator std::iterator<std::input_iterator_tag, T>. The iterator has also the alias rtAccelerationStructure<T>::iterator in complience with the STL iterators for containers. This base class iterator does not provide a meaningful implementation, but rather acts as a base for the iterators of other AS derived from the rtAccelerationStructure. When creating a new acceleration structure, developers should also subclass the rtAccelerationStructure<T>::iterator to create one suitable for the internal operation of the new AS (see the example implementation of the rtSoup AS).
